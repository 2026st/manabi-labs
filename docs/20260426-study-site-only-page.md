# 勉強サイトページの責務整理

## 変更内容

`app/routes/study.tsx` から記事表示機能を削除し、勉強サイト一覧専用ページに変更した。

- 記事データ取得 (`listArticles`) を削除
- 記事カード (`ArticleCard`) 表示を削除
- 空状態メッセージを `表示できる勉強サイトがありません` に変更

## 意図（なぜ必要か）

勉強サイトページは「有用サイトの追加と一覧表示」のための場所であり、
記事表示は責務外のため。

## 選択理由

- KISS: ページ責務を1つに限定
- YAGNI: 不要な記事読み込みを削除
- DRY: サイト表示ロジックのみを維持して誤解を防止

## 具体例

勉強サイトページでは、`基本情報技術者過去問道場` やユーザー追加サイトのみが表示され、
記事一覧は一切表示されない。
