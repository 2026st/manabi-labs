# Studyサイト追加ボタン実装

## 変更内容

`app/routes/study.tsx` に、勉強サイトを追加するUIを実装した。

- サイト名入力欄を追加
- URL入力欄を追加
- `サイトを追加` ボタンを追加
- 追加したサイトを勉強サイトページ内に一覧表示
- 一覧のリンクは `target="_blank"` で新しいページ（新規タブ）で開くように設定

## 実装意図

勉強サイトページ上で、ユーザーが任意の学習サイトを即時追加・参照できるようにするため。
記事作成画面に遷移せず、1ページ内で完結する操作にすることで学習導線を短くした。

## 選択理由

- KISS: 入力2項目 + 追加ボタン + 一覧表示の最小構成
- YAGNI: 現時点で未要求の編集/削除/永続化は実装しない
- DRY: リストUIは既存の `article-card` スタイルを再利用

## 具体例

- サイト名: `IPA`
- URL: `www.ipa.go.jp`

上記を入力して `サイトを追加` を押すと、URLは `https://www.ipa.go.jp` に正規化され、一覧に表示される。
