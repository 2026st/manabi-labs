# 追加ページ input サイズ調整

## 変更内容

- `app/styles/global.css` の `.input` の文字サイズを `24px` から `18px` に変更
- 同じく `.input` の余白を `10px` から `8px 10px` に変更

## 実装意図

- 追加ページ（`article.new`）のタイトル入力欄が視覚的に大きすぎたため、可読性を保ちつつ主張を抑えるために縮小
- `.textarea` など他要素は変更せず、影響範囲を入力欄のみに限定（KISS / YAGNI）

## 具体例

- 変更前: タイトル入力時の文字が見出し級（`24px`）で、フォーム全体に対して大きく見える
- 変更後: 本文入力欄とのバランスが取りやすい `18px` になり、フォームとして自然なサイズ感になる
